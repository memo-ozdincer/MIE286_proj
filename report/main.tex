% ─── 1. Introduction ─────────────────────────────────────────────
\section{Introduction}

Real-time performance feedback is a core component of interactive systems, from factory control panels to surgical interfaces to video games. In each case, users must respond quickly while maintaining accuracy, and the form that feedback takes---visual, auditory, or haptic---may shape how they navigate this speed--accuracy tradeoff (SAT). Wickens' Multiple Resource Theory (MRT) predicts that pairing a visual task with visual feedback should overload a single attentional channel, whereas auditory feedback distributes processing across modalities and reduces interference~[1,\,2]. Under this model, auditory feedback should yield faster, less disrupted responses.

We tested this prediction in a browser-based target-selection task and found a surprising reversal: participants were \emph{faster} under visual feedback, not slower. The visual condition used a gold/green/red border glow that created a salient speed incentive---participants could see whether they were ``fast'' or ``slow''---and this drove speed-chasing behavior that overrode MRT's predicted interference. This reversal, and what it reveals about the motivational dimension of feedback, is the central finding of this report.

We investigated three hypotheses:
\begin{enumerate}
  \item \textbf{H\textsubscript{1} (Speed):} Mean reaction time will differ between visual and auditory feedback conditions.
  \item \textbf{H\textsubscript{2} (Accuracy):} The faster condition will exhibit greater radial error, consistent with a speed--accuracy tradeoff.
  \item \textbf{H\textsubscript{3} (SAT structure):} The relationship between reaction time and radial error will differ between conditions---the SAT slope will be steeper under one modality.
\end{enumerate}


% ─── 2. Background ───────────────────────────────────────────────
\section{Background}

Wickens' Multiple Resource Theory posits that human attention operates through distinct pools organized by modality, processing stage, and response code~[1,\,2]. A visual pointing task paired with visual feedback should compete for the same attentional channel, whereas auditory feedback draws on a separate pool and should reduce interference. Cognitive Load Theory similarly predicts that cross-modal distribution of information lowers cognitive burden~[3].

Motor performance in pointing tasks is well characterized by Fitts' Law, which relates movement time to an index of difficulty defined by target distance and width~[4,\,5]. Our task holds target size constant and randomizes position, so performance differences should reflect cognitive processing of feedback rather than motor constraints alone. Consistent with Fitts' Law, we observed that edge-placed targets (requiring longer cursor travel) took 64\,ms longer than center targets (753\,ms vs.\ 689\,ms).

The speed--accuracy tradeoff is a fundamental constraint: faster responses are generally less accurate. Heitz~[6] emphasizes that the SAT is not a single parameter but a \emph{function} whose shape can change across conditions. We ask whether feedback modality changes this function. If visual feedback creates speed pressure, the SAT slope under visual should be steeper---each millisecond of haste costs more accuracy.

Prior work supports modality effects. Akamatsu et al.~[7] found that auditory feedback reduced movement time in a Fitts' Law task, and Sigrist et al.~[8] reviewed evidence that auditory feedback supports motor learning through cross-modal facilitation. However, these studies examined mean performance rather than the tradeoff structure. Whether feedback modality changes the SAT slope remains an open question.


% ─── 3. Methods ──────────────────────────────────────────────────
\section{Methods}

\subsection{Participants}
Fifty-five individuals (ages 18--24, university students) participated via convenience sampling. Covariates recorded: input device (Mouse, $n=17$; Trackpad, $n=27$; Other, $n=11$), gaming frequency (Never, $n=12$; Rarely, $n=19$; Weekly, $n=19$; Daily, $n=5$), and condition order (Visual-first, $n=27$; Auditory-first, $n=28$).

\subsection{Apparatus and Task}
The experiment was a browser-based application hosted on GitHub Pages (screenshots in Appendix~C). The task canvas measured 800$\times$500\,px. Circular targets (radius = 35\,px) appeared at random positions, and participants clicked on them as quickly and accurately as possible. Timing used the browser's \texttt{performance.now()} API (sub-millisecond resolution). A 400\,ms inter-trial interval separated targets.

% Interface screenshot is in Appendix C (Figure C.1)

\subsection{Independent Variable}
Feedback type was a within-subject variable with two levels:
\begin{itemize}
  \item \textbf{Visual:} A border glow---gold for fast hits ($<$600\,ms), green for slow hits ($\geq$600\,ms), and red with a shake animation for misses. Duration: 150\,ms.
  \item \textbf{Auditory:} Pure tones---880\,Hz for fast hits, 440\,Hz for slow hits, and 180\,Hz for misses. Duration: 200\,ms.
\end{itemize}
Both conditions conveyed \emph{identical three-level information} (fast hit, slow hit, miss). The on-screen HUD---trial counter, score, and streak---was identical in both conditions. This design addresses feedback from the proposal review, which noted an information asymmetry between conditions.

\subsection{Dependent Variables}
\begin{itemize}
  \item \textbf{DV\textsubscript{1}: Reaction time} (ms)---target onset to click.
  \item \textbf{DV\textsubscript{2}: Radial error} (px)---Euclidean distance from click to target center. A hit was defined as error $\leq$ 35\,px.
\end{itemize}

\subsection{Procedure and Design}
The study used a within-subject, counterbalanced design. After providing consent, participants completed: Instructions $\rightarrow$ 5 practice trials $\rightarrow$ Block~1 (30 or 60 trials) $\rightarrow$ Break $\rightarrow$ Block~2 $\rightarrow$ End screen. Condition order was randomized. Target size, canvas, ITI, and HUD were constant across conditions; target positions were randomized. A gamification system (scoring, particle effects, and a leaderboard with superlatives) was introduced mid-collection to boost engagement and recruitment.

\subsection{Potential Biases and Limitations}
Convenience sampling limits generalizability. Device variability introduced noise (mouse users were 142\,ms faster than trackpad users). Trial counts varied: 30 participants completed 30 trials/block and 25 completed 60; per-participant means normalize this. Audio environment was uncontrolled. The leaderboard introduced mid-collection may have shifted motivation: post-leaderboard participants were 77\,ms faster on average ($p = .007$), though accuracy was unchanged. Four participants completed the experiment more than once with different settings; each attempt was treated as a separate observation.


% ─── 4. Analysis ─────────────────────────────────────────────────
\section{Analysis}

\subsection{Data Cleaning and Aggregation}
Trials with RT $< 150$\,ms (anticipatory) or $> 8{,}000$\,ms (inattention) were excluded---3 of 4{,}800 trials (0.06\%). The remaining 4{,}797 trials were aggregated to per-participant means for each condition, yielding 55 paired observations.

\subsection{Assumption Checks}
Normality of within-subject differences was assessed with the Shapiro--Wilk test. RT differences were approximately normal ($W = 0.985$, $p = .704$), so a paired $t$-test was appropriate for H\textsubscript{1}. Error differences violated normality ($W = 0.892$, $p < .001$), so the Wilcoxon signed-rank test was used for H\textsubscript{2}. QQ plots are in Appendix~A.

\subsection{Hypothesis Tests}

\noindent\textbf{H\textsubscript{1} (Speed).}
Our original prediction, based on MRT, was that auditory feedback would produce faster responses by reducing intra-channel interference. The data showed the opposite---visual RT was numerically lower. We used a \emph{two-tailed} paired $t$-test because the observed direction contradicted our directional prediction:
\begin{align*}
  H_0 &: \mu_{\text{visual}} = \mu_{\text{auditory}} \\
  H_1 &: \mu_{\text{visual}} \neq \mu_{\text{auditory}}
\end{align*}

\noindent\textbf{H\textsubscript{2} (Accuracy).}
If visual feedback drives faster responses, the SAT predicts those responses should be less accurate. We used a \emph{one-tailed} Wilcoxon signed-rank test (normality violated):
\begin{align*}
  H_0 &: \text{median}(d_{\text{error}}) \leq 0 \\
  H_1 &: \text{median}(d_{\text{error}}) > 0, \quad d_{\text{error}} = \bar{e}_{\text{visual}} - \bar{e}_{\text{auditory}}
\end{align*}

\noindent\textbf{H\textsubscript{3} (SAT structure).}
We fit a multiple linear regression with an interaction term at the trial level:
\[
  \text{error} = \beta_0 + \beta_1 \cdot \text{RT} + \beta_2 \cdot \text{condition} + \beta_3 \cdot (\text{RT} \times \text{condition}) + \varepsilon
\]
A significant $\beta_3$ indicates different SAT slopes between conditions. We supplemented this with Pearson correlations and simple linear regressions per condition.

All tests used $\alpha = 0.05$. Cohen's $d$ is reported for paired comparisons. With $n = 55$ and the observed small effect sizes ($d \approx 0.15$), the study is underpowered for H\textsubscript{1} and H\textsubscript{2} (power $\approx 0.25$), increasing the probability of Type~II error.

\subsection{Inferential Statistics}

\textbf{H\textsubscript{1} (Speed).} A paired $t$-test found no significant difference in mean RT, $t(54) = -1.19$, $p = .240$, 95\% CI $[-25.3,\ 6.5]$\,ms, $d = -0.16$. We fail to reject $H_0$. The 9.4\,ms difference, while in the direction opposite to MRT's prediction, was not statistically reliable given high between-participant variability.

\textbf{H\textsubscript{2} (Accuracy).} A Wilcoxon signed-rank test found no significant difference in radial error, $V = 880$, $p = .180$ (one-tailed), $d = 0.15$. Visual error was 0.69\,px higher on average (median difference: 0.12\,px), but this was not significant. Accuracy was near ceiling ($\sim$95\% hit rate), limiting the range of detectable error differences.

\textbf{H\textsubscript{3} (SAT structure).} A multiple linear regression revealed a \emph{significant} RT$\times$condition interaction, $\beta_3 = -0.0083$, $t(4793) = -3.73$, $p < .001$, indicating that the SAT slope was steeper under visual feedback. The full model: $F(3,\, 4793) = 30.47$, $p < .001$, $R^2 = .019$. Simple regressions confirmed: the visual slope ($b = -0.014$, $R^2 = .018$) was more than twice as steep as the auditory slope ($b = -0.006$, $R^2 = .016$). Pearson correlations were $r = -.135$ ($p < .001$) for visual and $r = -.126$ ($p < .001$) for auditory. Figure~\ref{fig:sat} visualizes this difference.

Figure~\ref{fig:learning} shows mean RT across trial number within a block. The visual condition is consistently faster throughout, and the two curves are approximately parallel---the modality effect is stable across the block rather than emerging through practice.

\subsection{Spatial Click Patterns}

Plotting all clicks relative to target center (Appendix~B, Figure~B.6) revealed a systematic downward bias: on misses, participants clicked an average of 7.9\,px below the target and 2.8\,px to the left. This may reflect cursor hotspot offset or hand biomechanics. The miss cloud was slightly wider under visual feedback, consistent with higher error variance when speed-chasing.

\subsection{Secondary Findings}

\textbf{Tilt effect.} The probability of a miss following a previous miss was 12.2\% (26/213), compared with 4.2\% (187/4{,}474) following a hit---nearly three times higher, $\chi^2(1) = 28.38$, $p < .001$. This ``tilt'' effect, analogous to the poker term for emotionally-driven errors after a loss, suggests that errors temporarily disrupt attentional focus or motor calibration, creating cascading misses.

\textbf{Counterbalancing check.} An independent-samples $t$-test comparing overall mean RT between Visual-first ($M = 707$\,ms) and Auditory-first ($M = 713$\,ms) groups found no significant difference, $t(52.3) = 0.19$, $p = .850$, confirming that counterbalancing was effective.

\textbf{Covariates.} Mouse users were substantially faster than trackpad users (629\,ms vs.\ 771\,ms), reflecting the ergonomic advantage of a dedicated pointing device (Table~\ref{tab:covariates}). Daily gamers were the slowest group (770\,ms) but achieved the highest hit rate (97.3\%), suggesting a deliberate accuracy-over-speed strategy---a natural SAT operating at the individual level.

% ─── 6. Conclusion ───────────────────────────────────────────────
\section{Conclusion}

Neither mean RT nor mean error differed significantly between conditions (H\textsubscript{1}: $p = .240$; H\textsubscript{2}: $p = .180$). However, the \emph{structure} of the speed--accuracy tradeoff did differ significantly (H\textsubscript{3}: $p < .001$): the SAT slope under visual feedback was more than twice as steep as under auditory feedback.

The MRT prediction---that auditory feedback should reduce interference and produce faster responses---was not supported. Instead, the visual condition's gold/green border glow created a salient speed incentive: participants could \emph{see} whether they were ``fast'' (gold) or ``slow'' (green), and during data collection they were visibly frustrated at receiving green instead of gold. This speed-chasing behavior was absent in the auditory condition, where the tone distinction was less motivationally compelling. MRT's interference framework is therefore incomplete for predicting feedback effects: modality affects not only \emph{where} attentional resources are allocated but \emph{how aggressively} participants pursue speed.

The tilt effect ($\chi^2 = 28.38$, $p < .001$) adds further evidence: errors cascade, with a miss tripling the probability of a subsequent miss. This is consistent with attentional disruption or frustration-driven strategy shifts, not simple motor noise.

\textbf{Practical implications.} Visual feedback in safety-critical interfaces may inadvertently create speed pressure; auditory feedback may better preserve accuracy by decoupling the speed incentive from the visual task channel. Conversely, visual feedback may be preferred where speed is desirable.

\textbf{Limitations.} The convenience sample limits generalizability. Device variability introduced noise (mouse users 142\,ms faster than trackpad). Mixed trial counts were addressed via per-participant means. The leaderboard ($p = .007$) may have shifted motivation but was orthogonal to the within-subject comparison. Accuracy was near ceiling ($\sim$95\%), limiting detectable error differences. The visual border glow is inherently more salient than a tone; our results speak to feedback ``as deployed'' rather than modality \emph{per se}.

\textbf{Lessons learned.} Counterbalancing was essential ($p = .850$ confirmed no order effect). Trial independence cannot be assumed in sequential motor tasks. The most informative finding was the one we did not predict---future work should manipulate feedback salience independently of modality to disentangle the motivational and interference channels.

\section{R Code}

All analyses were conducted in R~4.5.3. The pipeline:

\begin{enumerate}
  \item \texttt{01\_load\_clean.R} --- Loads 55 CSVs, excludes RT outliers, aggregates per-participant means.
  \item \texttt{02\_descriptives.R} --- Summary tables by condition, device, gaming frequency.
  \item \texttt{03\_assumptions.R} --- Shapiro--Wilk tests and QQ plots.
  \item \texttt{04\_inferential.R} --- Paired $t$-test (H\textsubscript{1}), Wilcoxon (H\textsubscript{2}), multiple regression with interaction (H\textsubscript{3}).
  \item \texttt{05\_plots.R} --- All figures (spaghetti, density, SAT scatter, learning curve, click scatter, boxplots, covariates, diagnostics).
  \item \texttt{06\_secondary.R} --- Tilt effect, order check, variance, block comparison, leaderboard effect, spatial bias, per-participant SAT slopes.
\end{enumerate}

The master script \texttt{00\_run\_all.R} sources all scripts in sequence and is fully reproducible. Full code is in the \texttt{analysis/scripts/} directory.
